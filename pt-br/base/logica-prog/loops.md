# ♾️ Loops
## ✨ Visão Geral
Os Loops, também conhecidos como "laços de repetição", são estruturas que consistem em repetir um bloco de código diversas vezes, até que uma certa condição seja atendida.

No geral, e na maioria das linguagens, você vai se deparar com 3 tipos de Laços de Repetição.

### While
O laço While é a estrutura de repetição mais simples, caso uma condição seja verdadeira, seu bloco de código será executado.
```csharp
//Estrutura
while (condição) {
    //Código...
}
``` 
```csharp
//Exemplo

//Contador
int numQualquer = 8;

//Enquanto numQualquer for maior que 0, o bloco de código do while vai rodar
while (numQualquer > 0) {
    Console.Write(numQualquer); //Vai exibir: 876543210
    numQualquer--; //Decrementa (tira 1 do valor da variável) a variável
}
```
O While para de rodar caso a condição seja `false`, mas também caso um `break;` seja executado.
